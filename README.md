##  About The Website (Technical Details)

This repository contains the official digital storefront for Omni-Growth. It is built to deliver a premium, app-like experience directly in the browser.

###  Key Technical Features
*   **Custom SPA Architecture:** The site operates as a Single Page Application (SPA) using pure Vanilla JavaScript. It navigates between sections (Services, Projects, About, Contact) smoothly without ever reloading the browser.
*   **Smart WhatsApp CRM Integration:** The contact form captures user data (Name, Phone, Service, Project Details) and automatically formats it into a structured URL-safe string. It then redirects the user to WhatsApp, pre-filling the message directly to the studio's business number.
*   **Premium UI/UX:** Features a deep-blue corporate theme, modern glassmorphism (frosted glass navbar), hover state animations, and a fully responsive grid system.
*   **Zero Dependencies:** 100% lightweight codebase with no external libraries like React, Vue, or Bootstrap. 

###  Tech Stack
*   **Structure:** HTML5 (Semantic and accessible)
*   **Styling:** CSS3 (Custom variables, Flexbox, CSS Grid, Keyframe Animations)
*   **Logic:** Vanilla JavaScript (DOM manipulation, page routing, form data extraction)
*   **Typography:** Google Fonts (*Manrope* for bold headings, *Inter* for clean body text)
