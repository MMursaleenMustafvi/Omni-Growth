#  Omni-Growth | Premium Tech & Growth Studio

Welcome to the official repository for **Omni-Growth**, a premium, multi-disciplinary tech and growth studio portfolio. This website serves as the digital storefront for our services, including App/Web Development, AI/ML Solutions, Digital Marketing, FYP Consultancy, and Full Business Consulting.

##  Overview
The Omni-Growth website is designed to provide a seamless, app-like experience. It uses a custom-built **Single Page Application (SPA)** architecture using pure HTML, CSS, and Vanilla JavaScript. Users can navigate between sections (Services, Projects, About, Contact) without the page ever reloading, ensuring a lightning-fast and premium browsing experience.

##  Key Features
* **Custom SPA Routing:** Vanilla JavaScript handles page switching dynamically without browser reloads.
* **Premium UI/UX:** Features a deep-blue tech theme, glassmorphism effects (frosted glass navbar), and smooth hover states.
* **Smart WhatsApp Integration:** The contact form doesn't just send emails—it captures user data (Name, Phone, Service, Details) and automatically formats it into a structured WhatsApp message, sending the lead directly to the studio's phone.
* **Fully Responsive:** Optimized for all devices, from ultra-wide desktop monitors to small mobile screens.
* **High-Performance:** Lightweight codebase with zero external dependencies or heavy frameworks (No React/Vue required).

## Tech Stack
* **HTML5:** Semantic and accessible structure.
* **CSS3:** Custom CSS variables, CSS Grid, Flexbox, and CSS animations.
* **JavaScript (Vanilla):** DOM manipulation, page routing, and WhatsApp URL encoding.
* **Fonts:** Google Fonts (*Manrope* for headings, *Inter* for body text).
